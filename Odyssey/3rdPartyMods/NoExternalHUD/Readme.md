# No External HUD Mod v1.102.1 for Elite Dangerous 4.2.2.0

### by Fred89210<br><br>

**-- NEW for Elite Dangerous 4.2.2.0 --**<br><br>

**Release Notes:**
- Initial Release

-------------------------------------------------------------------------

**Description:**

This mod allows you to disable HUD elements that are located outside the ship.

-------------------------------------------------------------------------

**How to use:**

After installing the mod, start the game and press `ALT+F5` to disable the External HUD.
Press `ALT+F5` again to turn External HUD back on.
<br><br><br>

**Mod Hotkey can be changed:**

With EDHM_UI:

Go to `Main Menu -> 3PMods -> No External HUD Mod` and change the Key.<br><br>

Without EDHM_UI:

Edit `NoExternalHUD.ini` and change the line `Key = ALT F5`.
