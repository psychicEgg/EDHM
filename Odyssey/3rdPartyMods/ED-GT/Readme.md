# ED-GT v1.1.0 for Elite Dangerous 4.x

### by Blue Mystic<br><br>

## NOTE:

The ED-GT plugin is NOT a 3PMod for EDHM, but a plugin for EDHM_UI.

This plugin CANNOT work without EDHM_UI.<br>

Do NOT download this Plugin manually as it will NOT work, use EDHM_UI to install this Plugin and use it.<br>

-------------------------------------------------------------------------

**Description:**

Tweaks for the `GraphicsConfigurationOverride.xml` file.<br>

-------------------------------------------------------------------------

**How to Install:**

- Open EDHM_UI
- Go to `Main Menu -> 3PMods`
- Select `ED-GT`
- Install
