## EDHM v20.00 to v20.06 for Elite Dangerous v4.1.2.1 Patch Notes

### -- Ready for Elite Dangerous v4.1.2.1 (Trailblazer Update) --<br><br>

**EDHM v20.06:**
- EDHM Improvement Update
	- Surface Installation and Planetary Port Orange Icons on Planet Map now use the Galmap settings instead of a Green Color (y84 & xyzw85)
	- Color Filter for Main Menu Images now uses the settings from `Menus and Codex Image Filter` (w81 & xyzw181)
	- Ship Weapon Tri-Bar Hit Markers now use the Colors of associated Reticles rather than remaining Orange
	- OnFoot Weapon Loading Triangles in the Middle Panel now uses XML Matrix instead of Original Colors
	- Enabled Green Protection Color for the Green Squadron Icon in the Chat Panel
	- OnFoot Terminal Facility Door Image now displays better rendering
	- Manufacturer Orange Logos now use `Ship Sketches & Drawings` Color (w78 & xyzw146) instead of XML Matrix
	- Wing Neutral Target Icon at Top now use Radar Neutral Target Mode and Color instead of XML Matrix in order to avoid confusion in certain cases
	- Wing Shield at Top now use Custom Color (xyz128) in Custom Color Mode (y120) instead of Single Shield Color (xyz228)
	- Wing Middle Symbol now use XML Matrix when XML Mode (y120) is selected
	- Fix Wing Target Icon Color at Top which could display Black Icons with some XML Matrices
	- Fix `Target Ship Subcomponents` (w123 & xyzw125), now orange blocks should no longer randomly appear
	- `Target Ship Subcomponents` (w123 & xyzw125) now also applies to SRVs
	- Improved `Radar - Scanned by Neutral Target` Color (xyzw196)
	- Fix `Radar - Asteroids` option (w150)
	- Fix Highlighted Chat Icons which could display a solid block with certain color combinations
	- Fix Galmap Mission Completed Icon whose checkmark was not displaying the intended color
	- Fix Illegal Cargo Icon that was displaying a different color than the text
	- Fix an issue with the Codex Image Filter with some XML Matrices
	- Fix Borders of Commodities in Carrier Market Management while OnFoot
	- Fix Main Menu Colors when Connection Error (Video Background)
	- Fix Galmap Route Icon having a miscolored border
	- Fix Contamination on Target Rank Icons (w141)
	- Fix Shipyard Message Box when changing Ship
	- Fix FSS Red Signal (Before Energy Pulse)
	- Fix OnFoot Radar Green Target (Turret)
	- Fix Stinger Icon in Contacts Panel
	- Fix Cyan Bio Circle while OnFoot
	- Fix Scorpion Turret Icon
	- Fix Some Orange Banners
	
I've been working hard on this EDHM update to address long-standing bugs and imperfections.

Most of these issues are difficult to isolate, so I've also spent a lot of time testing the fixes.

However, some issues may have escaped my notice, so please report them to me on the [EDHM Discord](https://discord.gg/KTYgJegfYw).
